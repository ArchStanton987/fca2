# instructions spécifiques pour la création de battlemaps

- Les cartes doivent être des cartes type "battlemap" utilisables sur roll20.
- Elles ne doivent pas contenir d'annotation pour le MJ, il doit s'agir de la carte telle qu'elle sera affichée pour les joueurs.
- Pas de personnages représentés (les personnages sont ajoutés par la suite via des tokens)
- Les toits doivent être transparents pour révéler les détails des pièces (et que les pièces soient jouables)
- Avant de créer la carte, on récapitule les sections, les éléments de décor, leur utilité.
